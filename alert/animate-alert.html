Closing Alerts

To close the alert message, add a .alert-dismissible class to the alert container. Then add .btn-close and data-bs-dismiss="alert" to a link or a button element (when you click on this the alert box will disappear).

Success! This alert box could indicate a successful or positive action.
Info! This alert box could indicate a neutral informative change or action.
Warning! This alert box could indicate a warning that might need attention.
Danger! This alert box could indicate a dangerous or potentially negative action.
Primary! Indicates an important action.
Secondary! Indicates a slightly less important action.
Dark! Dark grey alert.
Light! Light grey alert.
